Introduction

This analysis explores the Pima Indians Diabetes Dataset containing medical measurements for 768 female patients. The data was collected by the National Institute of Diabetes and Digestive and Kidney Diseases and includes diagnostic measurements to investigate diabetes risk factors. The Pima Indian population has historically shown high diabetes prevalence, making this dataset valuable for understanding disease predictors. Data was originally collected in the 1990s from Pima Indian women aged 21+ living near Phoenix, Arizona.

Research Questions:

  1. How do glucose levels differ between diabetes groups?
  2. Which variables show the strongest differences?
  3. Does age affect diabetes prevalence?
  4. Which factor combinations appear most risky?

Question 1: Glucose Levels and Diabetes

Blood glucose is the primary diagnostic criterion for diabetes (ADA threshold: ≥126 mg/dL fasting).

Show code 
compare_groups <- function(data, variable, var_label) {
  summary_stats <- data %>%
    group_by(Outcome) %>%
    summarise(
      Mean = mean(get(variable), na.rm = TRUE),
      SD = sd(get(variable), na.rm = TRUE),
      Median = median(get(variable), na.rm = TRUE),
      Q25 = quantile(get(variable), 0.25, na.rm = TRUE),
      Q75 = quantile(get(variable), 0.75, na.rm = TRUE),
      .groups = "drop"
    )
  
  plot <- ggplot(data, aes(x = Outcome, y = get(variable), fill = Outcome)) +
    geom_violin(alpha = 0.6) +
    geom_boxplot(width = 0.2, alpha = 0.8) +
    labs(
      title = paste(var_label, "by Diabetes Status"),
      y = var_label,
      x = "Diabetes Status"
    ) +
    theme_minimal() +
    scale_fill_manual(values = c("#4ECDC4", "#FF6B6B")) +
    theme(legend.position = "none")
  
  return(list(stats = summary_stats, plot = plot))
}

This suggests diabetes diagnosis requires consideration of multiple factors beyond a single glucose measurement.

Question 2: Comparing Multiple Variables

To objectively compare different variables measured on different scales, we use Cohen’s d effect size (standardized difference between group means).

Effect size interpretation: Small (0.2), Medium (0.5), Large (0.8+)

Show code 
# Cohen's d calculation
cohens_d <- function(group1, group2) {
  mean_diff <- mean(group1, na.rm = TRUE) - mean(group2, na.rm = TRUE)
  pooled_sd <- sqrt((var(group1, na.rm = TRUE) + var(group2, na.rm = TRUE)) / 2)
  return(mean_diff / pooled_sd)
}

variables_to_compare <- c("Glucose", "BMI", "Age", "Pregnancies", 
                         "BloodPressure", "Insulin")

effect_sizes <- sapply(variables_to_compare, function(var) {
  no_diabetes <- diabetes_data[diabetes_data$Outcome == "No Diabetes", var]
  diabetes <- diabetes_data[diabetes_data$Outcome == "Diabetes", var]
  abs(cohens_d(no_diabetes, diabetes))
})

effect_size_results <- data.frame(
  Variable = variables_to_compare,
  Effect_Size = effect_sizes
) %>%
  arrange(desc(Effect_Size)) %>%
  mutate(Interpretation = case_when(
    Effect_Size >= 0.8 ~ "Large",
    Effect_Size >= 0.5 ~ "Medium",
    Effect_Size >= 0.2 ~ "Small",
    TRUE ~ "Negligible"
  ))

These rates are not representative of the general population.

Question 4: High-Risk Factor Combinations

We examine two-dimensional relationships to identify high-risk profiles. Smoothed trend lines show how diabetes probability varies across the risk space.

Show code 
create_risk_profile <- function(data, var1, var2, label1, label2) {
  ggplot(data, aes(x = get(var1), y = get(var2), color = Outcome)) +
    geom_point(alpha = 0.5, size = 2) +
    geom_smooth(method = "loess", se = FALSE, linewidth = 1.2) +
    labs(
      title = paste(label1, "vs", label2),
      x = label1,
      y = label2,
      color = "Outcome"
    ) +
    theme_minimal() +
    scale_color_manual(values = c("#4ECDC4", "#FF6B6B"))
}

p1 <- create_risk_profile(diabetes_data, "Glucose", "BMI", 
                         "Glucose (mg/dL)", "BMI")
p2 <- create_risk_profile(diabetes_data, "Age", "BMI", 
                         "Age (years)", "BMI")
p3 <- create_risk_profile(diabetes_data, "Glucose", "Age", 
                         "Glucose (mg/dL)", "Age (years)")
p4 <- create_risk_profile(diabetes_data, "Pregnancies", "Age", 
                         "Number of Pregnancies", "Age (years)")

grid.arrange(p1, p2, p3, p4, ncol = 2)

Findings: The smoothed curves (LOESS regression) show non-linear relationships between variables and help identify high-risk regions. The glucose-BMI plot shows strongest separation: individuals with both high glucose (>140 mg/dL) and high BMI (>35) have highest diabetes probability. The age-BMI combination also shows risk amplification at higher values. Pregnancy count shows weaker discriminative ability when combined with age, suggesting it adds limited information beyond age alone.

Summary and Conclusions

Key Findings:

  1. Glucose is the strongest single predictor (effect size: 1.07) but shows substantial group overlap, indicating diabetes classification requires multiple factors
  2. BMI is an important modifiable risk factor (effect size: 0.65)
  3. Age-related risk is pronounced, though the extremely high prevalence (35% overall, >50% in older groups) reflects this specific high-risk population
  4. Combined risk factors create multiplicative effects—high glucose + high BMI + advanced age represents the highest-risk profile
  5. Single-variable limitations: Even the strongest predictor cannot perfectly classify individuals, supporting the need for multivariable approaches (e.g., logistic regression models)

Dataset Representativeness: This dataset captures a population with genetic susceptibility and historically high diabetes rates. The 35% prevalence far exceeds the ~10% US national average, limiting generalizability. However, it remains valuable for studying risk factor relationships in high-risk populations.

Data Quality Considerations: Physiologically implausible zero values in BloodPressure and Insulin suggest missing data coded as zeros. Future analyses should properly handle these missing values.
